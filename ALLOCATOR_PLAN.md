## 〇、整体结构(改造后全景)

**一句话**:以风险平价为锚的全球大类配置[^a1];judgment 退为锚周围有上限的小幅倾斜[^a2];
执行层对标投资账本(Smart Portfolios)而非交易账本[^a3];本层只分配大类/市场间的风险,
不碰市场内部的选股与择时[^a4]。

**数据流(锚 → 下单)**:

```
[1] 分组树(进攻/防御 → 股票/加密、利率/实物信用)
     └ handcrafting 逐层等风险 → 各叶子风险权重 RW_i        ← 锚,零判断
[2] 横截面倾斜(仅市场间)
     └ 信号 =「估值 + 动量」组合,连续评分,偏离锚 ≤ ±5pp    ← 唯一保留的 judgment
[3] 风险权重 → 现金权重(逆波动率,σ 用混合 EWMA)
[4] 目标波动率校准(本轮纯现货,吃不满则如实报缺口)
[5] 月度新钱按缺口填补(只买少卖,不交易区控成本)
[6] 成本/税/换汇 → 落到 instrument_map 指定的 ETF/期货
```

**验证目标(不是求夏普)**:证明「横截面倾斜相对纯风险平价锚到底有没有正增量」。
**预期增量很小甚至为零 —— 那是成功的验证,不是失败**[^a5]。

## 一、改造后层级结构(旧 Layer 0/1/2 编号作废,以此为准)

### A. 锚层(战略风险分配,零 judgment)
- **砍** 原 Layer 0 的 CAPE 时间序列择时(总股票仓位)。进攻/防御总比例由锚顶层
  「进攻/防御风险比」**固定决定**(config 冻结),不再由 CAPE 择时[^b1]。
- 原 Layer 2 防御构成:**保留逆波动率 / 手工分组等风险,砍掉宏观四象限叠加**[^b2]。
- 锚算法:Carver **handcrafting**(分组逐层等风险,不依赖完整协方差优化);
  资产少且相关稳定的子组(政府债 + TIPS)内,可用 Qian **等风险贡献**做精细分配[^b3]。

### B. 倾斜层(唯一保留的 judgment,上锁)
- 保留原 Layer 1 横截面市场间倾斜,但:
  1. 偏离锚**硬上限**(各市场相对锚 ≤ ±5pp 风险)[^b4];
  2. 信号默认「**估值 + 动量**」组合,纯估值降为对照[^b5];
  3. **连续评分**,不用阈值开关[^b6]。

### C. 执行层(对标 Smart Portfolios,非 Systematic Trading)
- 新钱**优先填补欠配**资产(按正缺口分配)—— 再平衡主要靠注入,不靠卖出[^c1]。
- **不交易区为常驻核心机制**(非"规模大了才触发"):每个资产目标权重设容忍带,
  漂出带子才交易,且**只拉回带子边缘、不拉回正中**[^c2]。
- **带宽由成本/税驱动**:成本税越高带子越宽(A股/港股/美股各不同),越倾向不动[^c3]。
- "只买不卖"是宽带 + 新钱填补的**自然结果,非硬规则**;资产向上漂出带子(超配)时
  仍卖回边缘 —— 这是**失控保护,不是 timing**[^c4]。
- σ 用混合 EWMA;月度按缺口重算目标权重[^c5]。

### 砍 / 留一览(本次核心决策)

| 原层 | 内容 | 处置 | 脚注 |
|------|------|------|------|
| Layer 0 | CAPE 定总股票仓位择时 | **砍** | [^b1] |
| Layer 1 | CAPE 市场间横截面配置 | **留 + 上锁** | [^b4] |
| Layer 2 | 防御构成:逆波动率 | **留** | [^b2] |
| Layer 2 | 宏观四象限叠加 | **砍** | [^b2] |
| 风格轮动 | (原预留) | **不做** | [^d4] |

## 二、分组树(按风险来源,期货/ETF 是实现方式)[^d1]

```
总组合
├── 进攻 (risk-on)
│   ├── 股票:发达(US / 欧日) / 新兴(中国 / 其他新兴)
│   └── 加密(85:15 风险上限,见决策 1)
└── 防御 (risk-off)
    ├── 利率类:政府债 / 通胀债(TIPS)
    └── 实物/信用:黄金 / 信用(公司债 + 新兴债)
```

- 实现工具(ETF / 股指期货 / 国债期货 / 黄金期货)与分组位置**解耦**,
  config 里 `instrument_map` 单独配置,不影响树结构[^d2]。
- 树结构 + 顶层进攻/防御风险比,写进 config 带 hash 冻结[^d3]。

## 三、基准与归因

- **主基准** = 朴素分散版(handcrafting 锚 + 零倾斜 + 月度定投),零 judgment[^e1]。
- **对照基准** = 静态 60/40。
- **归因**:把「市场间倾斜」相对锚的增量**单独报告**;
  信号对照:纯估值 vs 估值+动量,证明组合是否更优[^e2]。

---

## 四、风险作为一等公民

- 设组合**目标波动率**(config)。输出实际波动率 vs 目标、各组**风险贡献**(非现金权重)[^f1]。
- 纯现货无杠杆会**吃不满**目标波动率 —— 如实报缺口,不假装达标[^f2]。
- **锚鲁棒性检验**:单独报告利率上行 / 债券熊市段(2022 及任何升息段)表现,
  拆解锚的夏普里有多少来自**债券久期红利**,防止「锚本身」就是债券牛市过拟合[^f3]。

## 五、组件审计(若无必要,勿增实体)

每个组件必须答出「理由 / 出处 / 如何验证」,否则砍。**「如何验证」栏即第三节归因要逐条回答的命题**。
**任何组件若验证不出正贡献(或贡献被成本/复杂度吃掉),默认砍;新增组件前先答这三栏,答不出不加。**

| 组件 | 理由(脚注) | 如何验证(否则砍) |
|------|-------------|---------------------|
| handcrafting 锚 | 抗估计误差[^b3] | vs 静态 60/40:风险调整后不更差即留 |
| 逆波动率(组内) | 等风险贡献[^f1] | 风险贡献分布是否真的拉平 |
| 估值+动量倾斜 | 二者负相关互补[^b5] | **倾斜增量 vs 锚;增量 ≤0 或不显著 → 砍**(注意样本长度,见[^e3]) |
| ——其中动量 | 治估值的"太早"[^b5] | 估值+动量 vs 纯估值,后者更优才留动量 |
| ±5pp 偏离上限 | 限制 judgment 下行[^b4] | 放大上限是否显著改善?否 → 维持收紧 |
| 连续评分 | 减信号跳变[^b6] | vs 阈值开关:换手与成本是否下降 |
| 不交易区/少卖 | 再平衡收益小、成本确定[^c2] | vs 定期强制再平衡:扣成本后净值差 |
| 定投节奏 | 匹配现金流现实[^c1] | (约束项,非优化项,不单独验证) |
| 黄金腿 | 危机/通胀对冲[^h1] | 加/不加黄金:尾部(2022)是否改善 |
| 加密腿 | 待验证:分散 or 仅 beta[^i1] | 2015 子样本:分散还是放大股票风险 |

> **剃刀的边界**:剃刀对准"说不清理由的复杂度",**不是"经过验证的必要分散"**。黄金/加密的另类腿
> 是补"股债同跌"尾部洞,非冗余;倾斜层若要砍,须先确认样本够长够多政体,别用剃刀名义犯
> "正确但太早"的反向错误[^e3]。

## 六、本层边界(立死,别越界)

本层只做「风险平价锚 + 市场间倾斜」在**大类/市场间**的风险分配。
**市场内部用什么指标、买什么产品、怎么择时,全部不在本层** —— 交给下游
「具体市场/具体产品」回测去证明。本层 equity leg 一律用宽基/因子 ETF 占位[^g1]。

---

## 七、已知局限(诚实写下)

- 危机对冲依赖**黄金 + 债券,不含能做空的趋势策略**,故「股债同跌」政体(2022 型)
  防御层可能整体失效 —— 本次不修复,已知[^h1]。
- 本层不证明任何市场内部的选股/择时 edge —— 那是下游回测的事[^g1]。

---

## 八、决策点(计划阶段确认,先别实现)

1. **加密**:已定进配置层、85:15 风险上限。前期无数据处理:全样本(1995+)**腿留空** +
   **2015 子样本单独评估**。**配置层加密与 OKX 交易层加密分开记账**(避免总敞口低估)[^i1]。
2. **杠杆变体**:本轮**只做纯现货无杠杆版**当基准;margin/期货放大到目标波动率 +
   多样化乘数的版本,**留接口、下一轮做**(复杂度高,先把无杠杆锚验证清楚)[^i2]。

## 九、硬约束(继续遵守)

- 参数**先冻结后验证**,严禁为改善回测结果调参(想调参让结果好看 → 停下问我)[^j1]。
- 成本/税/换汇建模;禁止前视;纯离线回测;**不宣称统计显著**[^j2]。
- **版本日志**:记录跑过的每个变体 + 参数集,避免事后只报好看的那个[^j3]。

## 十、流程

先建/改 data provider(stub → 实接)→ 改分组树与锚 → 改倾斜层(上锁 + 组合信号)
→ 改执行层(不交易区 + 新钱填补 + 超配卖回边缘)→ 写不变量测试(权重和=1、无前视、
成本守恒、新钱只填欠配、偏离 ≤ ±5pp)→ 出报告。**每阶段先给我看骨架再填。**
